Warehouse Management System 
**Description**
The Warehouse Management System (WMS) is a Java-based application designed to manage warehouse inventory operations efficiently. It allows users to perform inbound and outbound transactions, manage item stock levels, and track warehouse locations.
This project demonstrates object-oriented programming concepts, exception handling, and service-based architecture in Java.

**Features**
Add and manage warehouse items
Manage storage locations
Perform inbound transactions (add stock)
Perform outbound transactions (remove stock)
Automatic stock validation
Custom exception handling:
InvalidItemException
InvalidLocationException
InsufficientStockException
TransactionException
Display stock summary
Display location summary
